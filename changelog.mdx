## Recent Updates

Stay current with Alchemetryx enhancements. You receive automatic updates in your dashboard. Review changes below to understand impacts on your data analytics workflows.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Added real-time data streaming for live dashboards, enabling instant insights from UK and India sales pipelines.
- Introduced AI-powered anomaly detection to flag unusual patterns in SME financial datasets.

## Improvements

- Enhanced report export speeds by 40% for large CSV and PDF files.

## Bug Fixes

- Fixed dashboard refresh errors when handling datasets >1GB.
- Resolved authentication token expiry issues during long analysis sessions.

</Update>

## Upgrade Process

Follow these steps to ensure smooth updates across your deployments.

<Steps>
  <Step title="Check Compatibility" icon="check-circle">

    Review your current version in the dashboard settings.

    ```bash
    curl https://api.alchemetryx.com/v1/version \
      -H "Authorization: Bearer YOUR_TOKEN"
    ```

  </Step>
  <Step title="Backup Data" icon="database">

    Export critical datasets and reports.

  </Step>
  <Step title="Apply Update" icon="download">

    Updates deploy automatically. Monitor status at `https://dashboard.alchemetryx.com/updates`.

  </Step>
  <Step title="Verify Integrations" icon="settings">

    Test API endpoints and webhooks.

  </Step>
</Steps>

<Callout kind="tip">
  Enable email notifications in your account settings to receive instant alerts on breaking changes.
</Callout>
