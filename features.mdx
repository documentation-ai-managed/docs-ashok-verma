## Data Visualization and Reporting Tools

You generate compelling visualizations to uncover patterns in your data. Alchemetryx supports bar charts, line graphs, and heatmaps, pulling from CSV, SQL databases, or `https://api.example.com/data` endpoints.

<CodeGroup tabs="Python,JavaScript">
  ```python
  import alchemetryx as alc
  import pandas as pd

  df = pd.read_csv('sales_data.csv')
  viz = alc.Visualization(df)
  viz.bar_chart(x='month', y='revenue', title='Monthly Revenue')
  viz.export('revenue_report.png')
  ```
  ```javascript
  import { Visualization } from '@alchemetryx/sdk';

  const df = await fetch('https://api.example.com/sales').then(r => r.json());
  const viz = new Visualization(df);
  viz.barChart({ x: 'month', y: 'revenue', title: 'Monthly Revenue' });
  viz.export('revenue_report.png');
  ```
</CodeGroup>

<Callout kind="tip">
  Start with sample datasets from your dashboard to test visualizations quickly.
